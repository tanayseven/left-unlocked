Left Unlocked
=============

## An open source and simple script and webpage to remind or prank your colleagues to lock their computer when they leave their desk.

### Features
- [x] Simple and easy setup with static website
- [x] Website message
- [x] Shell script message
- [ ] Customizable message

### Usage

1. Open the browser on your colleague's computer
2. Simply visit the [website](https://leftunlocked.tanay.tech)
3. Either open the ![skull](./skull.svg) _Learn the Lesson_ 
4. Or click the ![copy](./copy.svg) _Copy Script Command_ button
5. Paste the command in the terminal and hit enter
